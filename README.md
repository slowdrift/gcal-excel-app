# Googleカレンダー → Excel 変換（週間日程表）

Googleカレンダー（またはICSファイル）の予定を、A4横の週間スケジュール表（15分単位・Excel形式）に変換するブラウザアプリです。

👉 **アプリを開く**: https://slowdrift.github.io/gcal-excel/

## 特徴

- **サーバー不要・完全ブラウザ完結**：予定データはお使いのブラウザ内だけで処理され、外部サーバーには送信されません（通信先はご自身のGoogleアカウントとフォント/ライブラリCDNのみ）
- 3つの読み込み方法：Google Calendar API（OAuth認証）／ICSファイル／ICS URL
- A4横・1週間=1ページの印刷レイアウト（自動縮尺）
- キーワードによる色分け（パステル7色）・文字色・強調（太字＋下線）
- プライバシー配慮の自動変換（人名→「来客あり」、指定キーワード→「予定あり」）
- 業務報告枠リストのコピー、処理ログによる変換内容の確認

## 使い方（Google Calendar API モード）

1. アプリを開き、画面の案内に従って Google Cloud Console で OAuth クライアントIDを作成
2. 「承認済みの JavaScript 生成元」にこのページのURLを追加
3. クライアントIDをアプリに貼り付けて認証 → カレンダー選択 → 期間指定 → Excelダウンロード

※ ICSファイル読み込みモードなら、認証設定なしですぐ使えます。

## プライバシー

- 予定データ・認証トークン・設定はすべてブラウザのローカルストレージにのみ保存されます
- 本アプリの提供者が利用者のデータを受け取ることはありません
